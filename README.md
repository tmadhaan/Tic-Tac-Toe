# Tic-Tac-Toe
A simple interactive tic tac toe game created using HTML, CSS, and JavaScript.

## Features
- Two player game
- Customizable player names
- Scores are continually tracked
- Restart function

## How to play
Both players are able to enter names, and then click on any empty spots on the grid to start. Player X starts first. First person with three in a row wins. Restart button allows players to keep playing while keeping track of the score.
